3. Projekt fo moduljai
----------------------

carbon-allowance
  Solidity es Hardhat alapu okosszerzodeses modul. Tartalmazza a CAC tokent,
  a registry contractot, a KYC logikat, a kvotakezelest es a piacteri contractokat.

carbon-dapp
  Next.js alapu fo webes felulet. Tartalmazza a wallet csatlakoztatast, profilt,
  admin oldalt, riportokat, surrender funkciot es piacteri oldalakat.

quota-calculator
  Kulon Next.js alkalmazas a kvotak demo szamitasahoz es lekerdezesehez.

tools
  Meresi, validalasi es benchmark segedeszkozok.


4. Lokalis futtatas roviden
---------------------------

4.1 Smart contractok

cd carbon-allowance
npm install
npm run compile
npm test

Lokalis Hardhat node inditasa:

npm run node

Masik terminalban lokalis deploy:

npm run deploy:all:local

Sepolia deploy:

npm run deploy:all:sepolia


4.2 Fo dApp

cd carbon-dapp
npm install
npm run dev

Lokalis URL:

http://localhost:3000

Build es lint ellenorzes:

npm run lint
npm run build


4.3 Kvotakezelo demo

cd quota-calculator
npm install
npm run dev

Lokalis URL:

http://localhost:4000

Build ellenorzes:

npm run build


5. Szerver es kornyezeti valtozok
---------------------------------

A projekt futtatasahoz kulon .env vagy .env.local fajlok szuksegesek.
Ezek biztonsagi okokbol nem kerulnek bele a publikus mellekletbe.

Fontosabb valtozok:

SEPOLIA_RPC_URL
PRIVATE_KEY
SERVER_PK
NEXT_PUBLIC_RPC_URL
NEXT_PUBLIC_CHAIN_ID
NEXT_PUBLIC_ALLOWANCE20_ADDRESS
NEXT_PUBLIC_REGISTRY_ADDRESS
NEXT_PUBLIC_MARKET_FIXED_ADDRESS
NEXT_PUBLIC_MARKET_OPEN_AUCTION_ADDRESS
NEXT_PUBLIC_MARKET_BUY_ORDER_ADDRESS
NEXT_PUBLIC_MARKET_BLIND_AUCTION_ADDRESS
NEXT_PUBLIC_MARKET_DUTCH_ADDRESS
NEXT_PUBLIC_MARKET_BUNDLE_ADDRESS
NEXT_PUBLIC_MARKET_OFFER_ADDRESS
PINATA_JWT
RPC_URL
CAC_ADDRESS
FACTOR_PER_M2

A privat kulcsokat, RPC kulcsokat es Pinata tokent nem szabad publikus repositoryba vagy
beadando ZIP-be feltolteni.


6. Melleklet tartalmanak megjegyzese
------------------------------------

A meretkorlat miatt a ZIP fajlbol kihagyhatoak az automatikusan ujrageneralhato vagy nagy
meretu fajlok:

node_modules/
.next/
artifacts/
cache/
.env
.env.local
.addr.*.local

Ezek npm install, build vagy deploy soran ujra letrehozhatok.


7. Rovid hasznalati leiras
--------------------------

1. A felhasznalo walletet csatlakoztat a fo dApp-ban.
2. A Profile oldalon vallalati profilt hoz letre es dokumentumot tolt fel.
3. Az admin feluleten a megfelelo operator jovahagyja a regisztraciot.
4. A kvotakezelo modul vagy a demo kvotaforras alapjan kvota rendelheto a felhasznalohoz.
5. A felhasznalo CAC tokent mintelhet, atadhat, surrenderelhet vagy piacteri funkciokban hasznalhatja.
6. A riport es receipt oldalak segitsegevel a tranzakciok visszakereshetok.
